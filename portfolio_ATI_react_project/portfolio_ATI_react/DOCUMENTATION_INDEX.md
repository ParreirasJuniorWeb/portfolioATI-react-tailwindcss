## 🎯 Como Usar Esta Documentação

### **Para Desenvolvedores Novos:**
1. Comece com [README.md](./README.md)
2. Leia [ARCHITECTURE.md](./ARCHITECTURE.md)
3. Estude [DEVELOPER_GUIDE.md](./DEVELOPER_GUIDE.md)
4. Consulte [COMPONENTS_REFERENCE.md](./COMPONENTS_REFERENCE.md)

### **Para Manutenção:**
1. Consulte [MAINTENANCE_GUIDE.md](./MAINTENANCE_GUIDE.md)
2. Use [TROUBLESHOOTING.md](./TROUBLESHOOTING.md) para problemas
3. Siga [UPGRADE_GUIDE.md](./UPGRADE_GUIDE.md) para atualizações

### **Para Deploy:**
1. Leia [DEPLOYMENT_GUIDE.md](./DEPLOYMENT_GUIDE.md)
2. Configure ambiente com [ENVIRONMENT_SETUP.md](./ENVIRONMENT_SETUP.md)
3. Siga [SECURITY_GUIDE.md](./SECURITY_GUIDE.md)

### **Para Testes:**
1. Siga [TESTING_GUIDE.md](./TESTING_GUIDE.md)
2. Consulte [TEST_RESULTS.md](./TEST_RESULTS.md)
3. Use [QA_CHECKLIST.md](./QA_CHECKLIST.md)
